popSSL

this is a universal SSL encryptor for pop3 traffic , it allows _ANY_ pop-3 mail reader to communicate via SSL to your server. No more POP-3 sniffers grabbing passwords :-)

Encryption is 128 bit world-wide with SSLeay !

the POP-3 server must support SSL encryption, for the server stuff try to find stunnel or change my code to support server mode certs :-)
